# Hardware

Questa cartella contiene i file hardware del progetto.

- `stl/` File STL per parti stampabili in 3D
- `schemi/` Schemi elettrici in PDF

## File principali
- `stl/orologio-coperchio-v01.stl`
- `stl/orologio-cornice-v01.stl`

## Note
- Mantieni i nomi file in minuscolo con trattini e versione (`-vNN`).
- Per nuove revisioni dei PDF incrementa la versione (`v02`, `v03`, ...).
