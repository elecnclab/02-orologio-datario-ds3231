# STL

Questa cartella contiene i file STL del progetto.

## Convenzione nome file consigliata
Formato:
`nome-parte-vNN.stl`

Esempi:
- `supporto-display-v01.stl`
- `cover-encoder-v02.stl`
- `distanziale-case-v01.stl`

## Regole pratiche
- Usa nomi in minuscolo.
- Usa il trattino `-` al posto degli spazi.
- Incrementa versione (`v01`, `v02`, ...) quando modifichi una parte.
- Non cancellare le versioni precedenti finché non hai validato la nuova stampa.
