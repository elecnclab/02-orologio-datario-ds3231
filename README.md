# Orologio Datario DS3231

Progetto Arduino per orologio con datario basato su:
- Arduino
- RTC DS3231
- Display a 7 segmenti
- Manopola/encoder

## Struttura cartelle
- `firmware/` Codice Arduino (.ino, librerie locali, configurazioni)
- `docs/` Documentazione tecnica, appunti, guide, BOM
- `media/` Foto del progetto, immagini per GitHub, screenshot
- `hardware/` Schemi elettrici, PCB, CAD/STL

## Stato iniziale
Struttura progetto creata e pronta per versionamento Git/GitHub.
