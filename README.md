# Orologio Datario DS3231

Progetto Arduino per orologio con datario basato su:
- Arduino Nano
- RTC DS3231
- Display a 7 segmenti
- Encoder rotativo con pulsante

## Struttura cartelle
- `firmware/` Codice Arduino (.ino, configurazioni, note firmware)
- `docs/` Documentazione tecnica e distinta base (BOM)
- `media/` Foto del progetto e immagini per GitHub
- `hardware/` STL, schemi elettrici e file hardware collegati

## Documentazione
- `docs/bom/bom.csv` Elenco componenti (BOM)
- `hardware/schemi/orologio-datario-schema-elettrico-v01.pdf` Schema elettrico

## Stato repository
Repository inizializzato, versionato e pubblicato su GitHub.
